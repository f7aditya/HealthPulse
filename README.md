# HealthPulse 🚑

A real-time healthcare resource coordination platform built using the MERN stack, enabling low-latency updates across multiple hospitals.

---

## 🚀 Features

- Real-time hospital resource updates (beds, blood, etc.)
- Multi-hospital data coordination
- Scalable backend architecture
- Secure API design
- Authentication system (JWT-based login)

---

## 🛠️ Tech Stack

- **Backend:** Node.js, Express
- **Database:** MongoDB (coming soon)
- **Real-time:** WebSockets (planned)
- **Caching:** Redis (planned)
